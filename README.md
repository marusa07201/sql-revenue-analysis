# SQL Revenue Analysis Project

## Project Description
This project demonstrates SQL skills for analyzing customer purchase data.

## Dataset
The dataset contains:
- order_id
- customer_id
- order_date
- revenue
- discount
- refund
- country

## Key Metrics Calculated
- Total Revenue
- Net Revenue
- Average Order Value (AOV)
- Monthly Revenue Trends
- Revenue by Country
- Refund Rate (%)
- Revenue Share by Country
- Customer Revenue Contribution (%)
- Running Total Revenue

## SQL Techniques Used
- Aggregate Functions (SUM, COUNT)
- NULLIF for safe division
- DATE_TRUNC
- GROUP BY
- ORDER BY
- LIMIT
- Window Functions (OVER)
- CTE (WITH)
